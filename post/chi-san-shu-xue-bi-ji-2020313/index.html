第一章 集合及其运算

1.1 集合的概念

1. 定义：

  • 集合： 一些可区分的互不相同的东西构成的整体称为集合， 简称集。一般用小写字母表示。

  • 元素： 集合的成员称为元素， 一般用大写字母表示。

  • 属于：一个元素 aaa 对于一个集合 AAA ，若 aaa 是 AAA 中的一个元素，则称 aaa 属于 AAA ，记作 a ∈ A{a\text{ }\in\text{ }A}a ∈ A 。

  • 不属于：若 aaa 不是 AAA 中的一个元素，则称 aaa 不属于AAA，记作a ∉ A{a\text{ } \notin \text{ }A}a ∈/​ A 。

  • 无穷集合：由无穷多个元素构成的集合称为无穷集合。

  • 有穷集合：由有限个元素构成的集合称为有限集合，或有穷集合。

  • 空集：不含任何元素的集合称为空集，记为 ∅{ \emptyset }∅ ，于是 ∅={}{ \emptyset = \left\{ \right\} }∅={} 。

  • 全集： 包含所考虑的目标内的所有元素的集合称为全集，常记为 UUU 。

2. 集合的表示方法：

  1. 枚举：将集合的元素逐一列举出来，元素之间用逗号“，”加以间隔，并用一对括号“{ }”括起来，这称为枚举法表示，或列举法表示。
    例：N={1,2,3.....}(其中N表示正整数){N= \left\{ 1,2,3..... \left\} \left( \text{其}\text{中}N\text{表}\text{示}\text{正}\text{整}\text{数} \right) \right. \right. }N={1,2,3.....}(其中N表示正整数)

  2. 性质描述法：用集合中元素的共同性质来刻画集合，这称为性质描述法。
    例：设 xxx 为某类对象的一般表示，p(x)p (x)p(x) 为关于 xxx 的一个命题，则用 {x∣P(x)}或{x:P(x)}{ \left\{ x \left| P \left( x \left) \left\} \text{或} \left\{ x:P \left( x \left) \right\} \right. \right. \right. \right. \right. \right. \right. }{x∣P(x)}或{x:P(x)} 表示具有性质 PPP 的那些元素构成的集合。
