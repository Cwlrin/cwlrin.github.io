命题逻辑

命题的概念

定义12.1 能表达判断的陈述句称为命题。

下列各句是命题：

  • 北京是中国的首都。
  • 月球上没有生物。
  • 222 是偶数而 333 是奇数。
  • 如果明天我有时间，那么我去科技馆参观。

下列各句不是命题：

  • x+y=4。（无确定的真假值）
  • 请你给我一支笔！（是祈使句而非陈述句）
  • 多么动听的音乐啊！（是感叹句而非陈述句）
  • 今天是星期一吗？（是疑问句而非陈述句）
  • 我正在说谎。（悖论）

说明：

  • 称能判断真假而不是可真可假的陈述句为命题。
  • 作为命题的陈述句所表达得的判断结果称为命题的真值。
  • 真值只取两个：真与假。真用 TTT 或 111 来表示；假用 FFF 或 000 来表示。
  • 一个命题，不真则假，不假则真，二者必居其一，且仅居其一。
  • 真值为真的命题称为真命题。真值为假的命题称为假命题。
  • 判断命题的步骤：
    • 是否为陈述句。
    • 是否有唯一的真值。

注意项：

  • 感叹句、疑问句、祈使句都不能称为命题。
  • 判断结果不唯一确定的陈述句不是命题。
  • 陈述句中的悖论不是命题。
  • 使用大写字母，或带下标的大写字母，或数字表示命题，例如：
    • PPP：雪是白的。
    • [1][1][1]：今天是3月7日。

定义12.2 表示命题的符号称为命题标识符。

定义12.3 如果一个命题标识符表示确定的命题，该标识符就称为命题常量，如果一个命题标识符只表示任意命题的位置标志，该标识符就称为命题变元。

定义12.4 命题变元可以表示任意命题，所以它不能确定真值，故命题变元不是命题。

定义12.5 当命题变元P用一个特定命题取代时，PPP 才能确定真值，这时称对 PPP 进行指派。

定义12.6 不能被分解成更简单句子的命题为简单命题或原子命题，由原子命题和联结词组合而成的命题称为复合命题。

联结词

否定联结词

  • 设 PPP 是一个命题，PPP 的否定是一个复合命题，记作 ¬P¬P¬P。如果 PPP 为真，则 ¬P¬P¬P 为假；如果 PPP 为假，则 ¬P¬P¬P 为真。

  • 例：PPP：他会开汽车。¬P¬P¬P：他不会开汽车。

  • 真值表：

    PPP ¬P¬P¬P
    1 0
    0 1

  • 否定可被视为一元运算，它的作用相当于自然语言中的“不”，“否”，“非”等，常把“¬¬¬”读作“非”。

合取联结词

  • 两个命题 PPP 和 QQQ 的合取是一个复合命题，记作 P∧QP∧QP∧Q 。当且仅当 PPP 与 QQQ 同时为真时，P∧QP∧QP∧Q 为真，在其他情况下，P∧QP∧QP∧Q 为假。

  • 真值表：

    PPP QQQ P∧QP∧QP∧Q
    1 1 1
    0 1 0
    1 0 0
    0 0 0

  • 合取可被视为二元运算，它的作用如同自然语言中的“与”、“且”等，常把 ∧∧∧ 读作“与”。

例：

  • PPP：今天下雨。
  • QQQ：命题下雨。
  • RRR：教室里有 101010 套桌椅。
  • 则 P∧QP∧QP∧Q： 今天和明天都下雨。
  • P∧RP∧RP∧R ：今天下雨且教室里有 101010 套桌椅。

说明：复合命题的真值只取决于各原子命题的真值，以及联结词的定义，而与原子命题的内容、含义无关。

析取联结词

  • 两个命题 PPP 和 QQQ 的析取是一个复合命题，记作 P∨QP∨QP∨Q 。当且仅当 PPP 与 QQQ 同时为假时，P∨QP∨QP∨Q 为假，在其他情况下，P∨QP∨QP∨Q 为真。

  • 真值表：

    PPP QQQ P∨QP∨QP∨Q
    1 1 1
    0 1 1
    1 0 1
    0 0 0

  • 析取可被视为二元运算，它的作用如同自然语言中的“或”，常把 ∨∨∨ 读作“或”。

例：

  • PPP：他会滑冰。
  • QQQ：他会游泳。
  • 则 P∨QP∨QP∨Q：他会滑冰或游泳。

“或”有两种，一种叫“可兼或”，另一种叫“不可兼或”。

  • PPP：小明在看电视。
  • QQQ：小明在睡觉。
  • 因看电视和睡觉，二者不能同时进行，故“小明在看电视或睡觉 ”中的“或”是“不可兼或”，不能用 P∨QP∨QP∨Q 表示“小明在看电视或在睡觉”。
  • 析取是“可兼或”。

条件联结词

  • 两个命题P和Q的条件是一个复合命题，记作 P→QP→QP→Q 。当且仅当 PPP 为真，QQQ 为假时，P→QP→QP→Q 为假，在其他情况下，P→QP→QP→Q 为真。PPP 称为前件，QQQ 称为后件。

  • 真值表：

    PPP QQQ P→QP→QP→Q
    1 1 1
    0 1 1
    1 0 0
    0 0 1

  • 条件可被视为二元运算，它的作用如同自然语言中的“如果 PPP，那么 QQQ”，“ QQQ 成立当 PPP ”，“PPP 成立仅当 QQQ”，“PPP 是QQQ 的充分条件”，“QQQ 是 PPP 的必要条件”。

例：

  • PPP：你给我发一个邮件。
  • QQQ：我会记住把资料寄给你。
  • 则 P→QP→QP→Q：如果你给我发一个邮件，那么我会记住把资料寄给你。

双条件联结词

  • 两个命题 PPP 和 QQQ 的双条件是一个复合命题，记作 P↔QP↔QP↔Q 。当且仅当 PPP 和 QQQ 的真值相同时，P↔QP↔QP↔Q 为真，PPP 和 QQQ 的真值不同时，P↔QP↔QP↔Q 为假。

  • 真值表：

    PPP QQQ P↔QP↔QP↔Q
    1 1 1
    0 1 0
    1 0 0
    0 0 1

  • 双条件可被视为二元运算，它的作用如同自然语言中的“当且仅当”，“充要条件”。

以上五个联结词又称逻辑运算符，优先级次序为：¬，∧，∨，→，↔¬，∧，∨，→，↔¬，∧，∨，→，↔。

原子命题是不包含任何联结词的命题，而至少包含一个联结词的命题是复合命题。

命题公式与恒等式

命题公式

定义12.7：命题公式，简称公式，归纳定义为：

  1. 单个命题变元和命题常元均是命题公式。
  2. 如果 AAA 和 BBB 是命题公式，则 (¬A),(A∧B),(A∨B),(A→B),(A↔B)(¬A), (A∧B), (A∨B), (A→B),(A↔B)(¬A),(A∧B),(A∨B),(A→B),(A↔B) 均是命题公式。
  3. 只有有限次地应用(1)和(2)所得到的包含命题常量、命题变元、联结词和圆括号的符号串是命题公式。

在命题公式中，命题变元不是命题，它没有真值，因此，命题公式的值不确定。

命题指派

如果给命题变元一个确定的指派，此时命题公式便成了命题，它才有确定的真值。

定义12.8：设 AAA 是一个命题公式，P1、P2、P3、…、PnP_1、P_2、P_3、…、P_nP1​、P2​、P3​、…、Pn​ 是出现在公式 AAA 中的所有命题变元，指定 P1、P2、P3、…、PnP_1、P_2、P_3、…、P_nP1​、P2​、P3​、…、Pn​ 一组真值，则这组真值称为 AAA 的一个指派或解释。

例：命题公式 (P∧Q)∨S(P∧Q)∨S(P∧Q)∨S 的一个指派为：P←T，Q←F，S←TP←T，Q←F，S←TP←T，Q←F，S←T，在此指派下，上述命题公式变成了命题，其值为 TTT。

如果一个命题公式 AAA 中有 nnn 个变元，则 AAA 共有 2n2n2n 种指派。

命题符号化

把一个用文字叙述的命题相应地写成由命题标识符、联结词和括号表示的命题公式，称为命题符号化。

注意：

  1. 确定给定句子是否为命题
  2. 句子中的连接词是否为命题联结词
  3. 要正确地表示原子命题并适当选择命题联结词

命题符号化的步骤

  1. 找出文字叙述的命题中的所有原子命题
  2. 用命题标识符来表示每个原子命题
  3. 根据句子中的连接词的语义，选择合适的命题联结词
  4. 使用命题联结词联结命题标识符，得到命题公式

例12.8：将自然语言描述的命题：“小张既聪明，又勤奋，所以他成绩好。”进行符号化。

解：

  • 设 PPP：小张聪明，QQQ：小张勤奋，RRR：小张成绩好
  • 命题符号化：(P∧Q)→R(P∧Q)→R(P∧Q)→R

等价置换法

定义12.10：如果 XXX 是命题公式 AAA 的一部分，且 XXX 本身也是一个命题公式，则称 XXX 为 AAA 的子公式。

定理12.1： 设 XXX 是命题公式 AAA 的子公式，且 X⇔YX⇔YX⇔Y ，如果将 AAA 中的 XXX 用 YYY 来置换，所得到的公式 BBB 与公式 AAA 等价，即 A⇔BA⇔BA⇔B。

证明：因为 X⇔YX⇔YX⇔Y，所以在相应变元的任一种指派情况下，XXX 与 YYY 的真值相同，故以 YYY 取代 XXX 后，公式BBB 与公式 AAA 在相应的指派情况下，其真值亦相同，故 A⇔BA⇔BA⇔B。

满足上述定理条件的置换称为等价置换（或等价代换）。

对偶原理

定义12.11：设有命题公式 AAA，其中仅有联结词 ¬、∧¬、∧¬、∧ 和 ∨∨∨。在 AAA 中将 ∧、∨、F、T∧、∨、F、T∧、∨、F、T 分别相应的换以∨、∧、T、F∨、∧、T、F∨、∧、T、F 而得到公式 A∗A*A∗，称 A∗A*A∗ 为 AAA 的对偶式。

例：¬P∨(Q∧R)¬P∨(Q∧R)¬P∨(Q∧R) 的对偶式为 ¬P∧(Q∨R)¬P∧(Q∨R)¬P∧(Q∨R)

定理12.2：设命题公式 AAA 和 A∗A^*A∗ 是对偶式，P1,P2,...,PnP_1,P_2,...,P_nP1​,P2​,...,Pn​ 是出现在 AAA 与 A∗A*A∗ 中的所有命题变元，则 ¬A(P1,P2,...,Pn)⇔A∗(¬P1,¬P2,...,¬Pn)，A(¬P1,¬P2,...,¬Pn)⇔¬A∗(P1,P2,...,Pn)¬A(P_1,P_2,...,P_n)⇔A^*(¬P_1,¬P_2,...,¬P_n)，A(¬P_1,¬P_2,...,¬P_n)⇔¬A*(P_1,P_2,...,P_n)¬A(P1​,P2​,...,Pn​)⇔A∗(¬P1​,¬P2​,...,¬Pn​)，A(¬P1​,¬P2​,...,¬Pn​)⇔¬A∗(P1​,P2​,...,Pn​)。

定理12.3：设 P1,P2,...,PnP_1,P_2,...,P_nP1​,P2​,...,Pn​ 是出现在命题公式 AAA 和 BBB 中的所有命题变元，如果 A⇔BA⇔BA⇔B，则 A∗⇔B∗A^*⇔B^*A∗⇔B∗。

重言式与蕴含式

重言式

定义12.12：在所有指派下，其值永为真的命题公式，称为重言式，或永真式；其值永为假的命题公式，称为矛盾式，或永假式；不是永假式的命题公式称为可满足式。

定义12.13：如果公式 AAA 在指派 XXX 下，其值为真，则称 AAA 是可满足的，或称 XXX 满足了 AAA 。

永真式的特点：

  • 永真式的否定为永假式，永假式的否定为永真式。
  • 两个永真式的合取式、析取式、条件式和双条件式都仍是永真式。

定理12.4：如果命题公式 PPP 和 QQQ 均为永真式，则 ¬P¬P¬P 为永假式，而 P∧Q，P∨Q，P→QP∧Q，P∨Q，P→QP∧Q，P∨Q，P→Q 和 P↔QP↔QP↔Q 均为永真式。

定理12.5：命题公式 AAA 和 BBB，A⇔BA⇔BA⇔B 的充要条件是 A↔BA↔BA↔B 为永真式。

恒等关系的性质

  • 与 AAA 永等值，所以 A⇔AA⇔AA⇔A，故恒等关系是自反的；
  • 如果 A⇔BA⇔BA⇔B，即 AAA 在任意指派下，与 BBB 等值，当然 BBB 与 AAA 亦等值，所以 B⇔AB⇔AB⇔A，故恒等关系是对称的。

定理12.6：命题公式 A，BA，BA，B 和 CCC，如果 A⇔BA⇔BA⇔B 且 B⇔CB⇔CB⇔C，则 A⇔CA⇔CA⇔C。故恒等关系是传递的。

恒等关系是一个等价关系。

蕴含式

定义12.14：设 AAA 和 BBB 是命题公式，在任意指派下，如果 AAA 为真，则 BBB 必为真，则称 AAA 永真蕴含 BBB，或简称 AAA 蕴含 BBB，记为 A⇒BA⇒BA⇒B。

定义中只强调了 AAA 为真时，BBB 必为真，而并不关心 AAA 为假时，BBB 是否为真。定义本身提供了一种证明 A⇒BA⇒BA⇒B 的方法。

间接法

定理12.15：设 SSS 是前提公式的集合，BBB 和 CCC 是两个公式。如果从 S∧BS∧BS∧B 可演绎出 CCC ，则从 SSS 可演绎出 B→CB→CB→C。此定理常称为 CPCPCP 规则。BBB 作为附加前提。

反证法

定义12.24：如果一组公式 A1,A2,...,AnA_1,A_2,...,A_nA1​,A2​,...,An​ 的合取永假，则称 A1,A2,...,AnA_1,A_2,...,A_nA1​,A2​,...,An​ 不相容，反之则相容。
