第一节 向量及其线性运算

1. 向量的概念

向量：既有大小又有方向的量，又称矢量。

在数学上，常用一条有方向的线段，即有向线段来表示向量。有向线段的长度表示向量的大小，有向线段的方向表示向量的方向。

以 AAA 为起点、 BBB 为终点的有向线段所表示的向量记作 AB→{ \overrightarrow {AB}}AB。有时也用一个黑体字母（书写时，在字母上面加箭头）来标识向量，例如 aaa、rrr、vvv、FFF 或 a→{ \overrightarrow {a}}a 、 r→{ \overrightarrow {r}}r、v→{ \overrightarrow {v}}v、F→{ \overrightarrow {F}}F 等。

自由向量：与起点无关的向量。

如果两个向量 aaa 和 bbb 的大小相等，且方向相同，则向量 aaa 和 bbb 是 相等的 ，记作 a=b{a = b}a=b。这就是说，经过平行移动后能完全重合的向量是相等的。

向量的大小：向量的大小叫做向量的模，向量 AB→{ \overrightarrow {AB}}AB、aaa、 a→{ \overrightarrow {a}}a 的模依次记为 ∣AB→∣{ \left| \overrightarrow {AB} \right| }∣∣∣​AB∣∣∣​ 、∣a∣{ \left| {a} \right| }∣a∣ 、∣a→∣{ \left| \overrightarrow {a} \right| }∣∣∣​a∣∣∣​。

单位向量：模等于1的向量叫做单位向量。

零向量：模等于零的向量叫做零向量，记作 000 或 0→{ \overrightarrow {0}}0。零向量的起点和终点重合，它的方向可以看做是任意的。

设有两个非零向量 a，ba，ba，b ，任取空间一点 OOO ，作 OA→=a{ \overrightarrow {OA}=a}OA=a，OB→=b{ \overrightarrow {OB}=b}OB=b ，规定不超过 π{ \pi }π 的 ∠AOB{ \angle AOB}∠AOB（设 φ=∠AOB,0≤φ≤π{\varphi = { \angle AOB}, 0{ \le }\varphi { \le }\pi}φ=∠AOB,0≤φ≤π）称为 向量 aaa 与 bbb 的夹角 ，记作 (a,bᨈ){ ( \mathop{{a,b}}\limits^{ᨈ} ) }(a,bᨈ​) 或 (b,aᨈ){ ( \mathop{{b,a}}\limits^{ᨈ} ) }(b,aᨈ​) ，即(a,bᨈ)=φ{{ ( \mathop{{a,b}}\limits^{ᨈ} ) = \varphi }}(a,bᨈ​)=φ。如果向量 aaa 与 bbb 中有一个是零向量，规定它们的夹角可以在 0 到 π{\pi}π 之间任意取值。

由于零向量与另一个向量的夹角可以在 000 到 π{\pi}π 之间任意取值，因此可以认为零向量与任何向量都平行，也可以认为零向量与任何向量都垂直。

向量加法规定如下：

向量的加法符合下列运算规律：

  • 交换律：a+b=b+a{a + b = b + a}a+b=b+a
