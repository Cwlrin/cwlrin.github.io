有向路与有向圈

有向通道

定义：设 D=(V,A)D=(V,A)D=(V,A) 是一个有向图，一个点弧交替序列 P=v0x1v1x2v2…xmvmP=v_0x_1v_1x_2v_2…x_mv_mP=v0​x1​v1​x2​v2​…xm​vm​，满足 xi=(vi−1,vi)x_i=(v_i-1,v_i)xi​=(vi​−1,vi​)，则称 PPP 为从 v0v_0v0​ 到 vmv_mvm​ 的有向通道，简记为 v0v1v2…vmv_0v_1v_2…v_mv0​v1​v2​…vm​。v0v_0v0​ 称为 PPP 的起点，vmv_mvm​ 称为 PPP 的终点。mmm 为通道的长度。当 v0=vmv_0=v_mv0​=vm​ 时，称 PPP 为有向闭通道。包含 DDD 的所有顶点的通道(闭通道)称为 DDD 的生成通道(闭通道) 。

说明：有向通道和有向闭通道中，顶点和弧均可重复出现，其长度是按弧出现的次数计数弧的条数。

有向迹和有向闭迹

定义：若有向图中有一条有向通道上的各弧互不相同，则称该通道为有向迹。若一条有向闭通道上的各弧互不相同则称该闭通道为有向闭迹。

有向路、有向圈

定义：若有向图中有一条有向通道上的各顶点互不相同，则称该通道为有向路。若一条有向闭通道上的各顶点互不相同则称该闭通道为有向圈或有向回路。

弱通道：如果不考虑弧的方法，顶点 uuu 与 vvv 间有一条无向通道，称之为弱通道。如果弱通道的起点和终点相同，称该弱通道为闭弱通道。如果弱通道上的各顶点互不相同，称该弱通道为弱路。如果闭弱通道上的各顶点互不相同，称该闭弱通道为弱圈。

哈密顿有向图：包含有向图 DDD 的所有顶点的有向圈称为 DDD 的生成有向圈。有生成有向圈的有向图称为哈密顿有向图，生成有向圈也叫有向哈密顿圈。

定理：设 D=(V,A)D=(V,A)D=(V,A) 是一个有向图，BBB 是 DDD 的邻接矩阵，则 DDD 中从顶点 viv_ivi​ 与顶点 vjv_jvj​ 的长度为 lll 的有向通道的数目等于 BlB^lBl 的元素 bij(l)b_{ij}^{(l)}bij(l)​ 的值。

顶点间可达和可达矩阵

定义：设 D=(V,A)D=(V,A)D=(V,A) 是一个有向图，vi,vj∈Vv_i,v_j\in Vvi​,vj​∈V，如果在 DDD 中存在从 viv_ivi​ 到 vjv_jvj​ 的有向路，则称从 viv_ivi​ 到 vjv_jvj​ 可达。特别地，任何顶点 viv_ivi​ 到自身是可达的。

定义：设有向图 D=(V,A)D=(V,A)D=(V,A) 的顶点集 V={v1,v2,…,vn}V=\{v_1,v_2,…,v_n\}V={v1​,v2​,…,vn​}，n×nn×nn×n 矩阵 P=(pij)P=(p_{ij})P=(pij​) 称为 DDD 的可达矩阵，其中

pij={1(从vi到vj可达)0(从vi到vj不可达)p_{ij}=\begin{cases}1\qquad(从 v_i到v_j可达)\\0\qquad(从v_i到v_j不可达)\end{cases}pij​={1(从vi​到vj​可达)0(从vi​到vj​不可达)​

说明：BBB 为有向图 DDD 的邻接矩阵，则 DDD 的可达矩阵 P=I∨B∨B2∨…∨Bn−1P=I\lor B\lor B^2\lor… \lor B^{n-1}P=I∨B∨B2∨…∨Bn−1。

强连通有向图

定义：在有向图 DDD 中，如果任意两个不同的顶点都是相互可达的，则称 DDD 是强连通的；如果任意两个不同的顶点，至少从一个顶点到另一个顶点是可达的，则称 DDD 是单向连通的；如果两个不同的顶点间有一条弱路，则称 DDD 是弱连通的，简称为连通的。

说明：强连通必是单向连通，单向连通必是弱连通，反之不成立。

定义：设 DDD 是一个有向图，DDD 的极大强连通子图，称为 DDD 的强分图；DDD 的极大单向连通子图，称为 DDD 的单向分图；DDD 的极大弱连通子图，称为 DDD 的弱分图。

定理：有向图的每个顶点都恰处在一个强分图(弱分图)之中。

定理：一个没有有向圈的有向图中至少有一个出度为 0 的顶点。

说明：在有向图的实际应用中，一个有向图是否有有向圈起决定性作用。

定理：有向图中没有有向圈当且仅当它的每一条有向通道都是有向路。

定理：有向图 D=(V,A)D=(V,A)D=(V,A) 中有有向圈的充分必要条件是 DDD 有一个子图 D1=(V1,A1)D_1=(V_1,A_1)D1​=(V1​,A1​)，使得 ∀v∈V1\forall v \in V_1∀v∈V1​，有 id(v)>0id(v)>0id(v)>0 且 od(v)>0od(v)>0od(v)>0。

有向树与有序树

有向树

定义：一个没有弱圈的弱连通的有向图称为有向树。如果一个有向图的每个连通分支是有向树，则称该有向图为有向森林。

说明：一个有向树去掉它的弧的方向时，得到的无向图是一棵无向树。

有向树弧的条数 m=顶点数n−1m=顶点数n-1m=顶点数n−1。

定义：如果一棵有向树恰有一个顶点的入度为 0 ，其余所有顶点的入度均为 1 ，则称该有向树为有根树。其中入度为 0 的顶点称为树根，出度为 0 的顶点称为树叶，出度非零的顶点称为分枝点或内顶点。

说明：

  • 有根树的图解通根顶点画在上面，弧的方向朝下。
  • 有根树从树根到每个顶点都恰有一条有向路。

定义:有根树的反向树称为入树。

说明：

  • 入树是没有弱圈的弱连通的有向图。
  • 入树恰有一个顶点的出度为 0 ，称为汇；其余所有顶点的出度均为 1 。

定义：设 D=(V,A)D=(V,A)D=(V,A) 是有根树，u,v,wu,v,wu,v,w 是有根树的顶点。如果 (u,v)∈A(u,v)\in A(u,v)∈A，则称 vvv 是 uuu 的儿子，而 uuu 是 vvv 的父亲，同一父亲的不同儿子称为兄弟。如果从 uuu 到 www 有有向路，则称 www 是 uuu的子孙，而 uuu 是 www 的祖先。由顶点 uuu 和它的所有子孙导出的子图称为以 uuu 为根的子树。从树根到顶点 uuu 的有向路的长度称为顶点 uuu 的层数。从根到树叶的最大层数称为有根树的高。

说明：有根树的任何子树必是有根树。

有序树

定义：如果将有根树每个顶点的儿子顶点都规定了次序，则称该有根树为有序树。如果一个有向图的每个连通分支是有序树，则称该有向图为有序森林。

说明：有序树的图解中规定每个顶点的儿子是从左向右依次排序的。

定义：设 TTT 为一棵有序树，如果 TTT 的每个顶点的出度 ≤m≤m≤m ，则称 TTT 为 mmm 元有序树。一棵 mmm 元有序树 TTT 的每个顶点的出度不是 0 就是 mmm，则称 TTT 为正则 mmm 元有序树。

说明：二元有序树简称二元树，一个分枝点的儿子被区分为左儿子或右儿子。

判定树与比赛图

判定树

有序树的一个重要应用是判定。

例：八硬币问题：有八枚硬币 a,b,c,d,e,f,g,ha,b,c,d,e,f,g,ha,b,c,d,e,f,g,h，有一枚是假的，外表与真硬币一样但重量与真硬币不一样。试用天平称出哪枚硬币是假的，并指出假硬币比真硬币轻还是重。要求比较次数最少。

  1. 第一步取六枚硬币 a,b,c,d,e,fa,b,c,d,e,fa,b,c,d,e,f 进行测量，若 a+b+c<d+e+fa+b+c<d+e+fa+b+c<d+e+f，则假硬币就在这六枚硬币中，g,hg,hg,h 为真硬币。
  2. 第二步取六枚硬币中的四枚 a,b,d,ea,b,d,ea,b,d,e 进行测量，若 a+d<b+ea+d<b+ea+d<b+e，则 c,fc,fc,f是真的，b,db,db,d 也是真的，aaa 和 eee 中有一个是假的。
  3. 第三步取一枚真硬币，如 bbb，测量 bbb 和 aaa，若 b=ab=ab=a，则 eee 是假的且 eee 比真硬币重，否则 aaa 是假的且 aaa 比真硬币轻；
  4. 若 a+d=b+ea+d=b+ea+d=b+e，则 a,b,d,ea,b,d,ea,b,d,e 是真的，ccc 和 fff 中有一个是假的，重复第三步便可区分c和f哪个是假的。
  5. 若 a+d>b+ea+d>b+ea+d>b+e，情况与 a+d<b+ea+d<b+ea+d<b+e 类似可区分哪一枚是假硬币。
  6. 一步测量时 a+b+c>d+e+fa+b+c>d+e+fa+b+c>d+e+f，过程与 a+b+c<d+e+fa+b+c<d+e+fa+b+c<d+e+f 类似。
  7. 第一步测量时 a+b+c=d+e+fa+b+c=d+e+fa+b+c=d+e+f，ggg 和 hhh 有一个是假的。

上述过程 可以用判定树表示。

判定树高度为3，所以只要称3次就能找到哪一枚硬币是假的，并指出假硬币比真硬币轻还是重。

判定树的重要应用：计算排序算法的速度。

比赛图

比赛图：给定的选手进行循环比赛时，每一对选手都互相比赛并产生一个胜利者，比赛没有平局。用点表示选手，对于每一对点，从胜者指向负者画一条弧，则产生一个比赛图。

定义：一个比赛图是一个定向完全图，它的任何两顶点间有且仅有一条弧。

定理：每一个比赛图有条生成有向路。

说明：每个比赛图中必有哈密顿路。
