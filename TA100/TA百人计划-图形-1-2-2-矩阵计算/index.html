# 什么是线性方程？

线性方程，顾名思义，呈线性的方程，一条线下来，比如我们中学时期学过的一次方程就是一个线性方程，线性方程具有可加性，比例性，即常数倍 kkk，用数学函数表达方式就是

可加性：

f(x1+x2)=f(x1)+f(x2)2x+3y=1f(x_1+x_2)=f(x_1)+f(x_2)\newline 2x+3y=1 f(x1​+x2​)=f(x1​)+f(x2​)2x+3y=1

比例性：

f(kx)=kf(x)x+y=22x+2y=4f(kx)=kf(x)\newline x+y=2\newline 2x+2y=4 f(kx)=kf(x)x+y=22x+2y=4

但是并不是所有方程都是线性方程，比如我们学过的三角函数，xxx 元 xxx 次方程就不是线性方程，此类被称为非线性方程：

x+y2=5sin⁡x+cos⁡x=1x+xy=3x+y^2=5\newline \sin x+\cos x = 1\newline x+xy=3 x+y2=5sinx+cosx=1x+xy=3

# 什么是线性空间

直线变换后依然是直线，并且等比坐标原点保持不变

# 什么是非线性空间

空间扭曲，不是等距，坐标原点有位移

# 矩阵的加减法

两个矩阵的行数，列数都相等时，就称他们是同型矩阵

[3512]+[1304]=[3+15+31+02+4]=[4816]\begin{bmatrix}3&5\\\\1&2\end{bmatrix}+\begin{bmatrix}1&3\\\\0&4\end{bmatrix}=\begin{bmatrix}3+1&5+3\\\\1+0&2+4\end{bmatrix}=\begin{bmatrix}4&8\\\\1&6\end{bmatrix} ​31​52​​+​10​34​​=​3+11+0​5+32+4​​=​41​86​​

# 矩阵与矩阵之间的乘法

首先并不是所有矩阵都能相乘，矩阵相乘需要满足一定的合法性

m×nm \times nm×n 的矩阵只能与 n×pn \times pn×p 的矩阵相乘

相乘后的矩阵大小为 m×pm \times pm×p

举例：

\begin{bmatrix}2&1\\\\0&3\\\\5&4\end{bmatrix}\times\begin{bmatrix}7&6\\\\8&9\end{bmatrix}=\begin{bmatrix}2\*7+1\*8&2\*6+1+9\\\\0\*7+3\*8&0\*6+3\*9\\\\5\*7+4\*8&5\*6+4\*9\end{bmatrix}

c_{ij}=a_{i\*}\cdot b_{\*j}\ (i=1,2,3...,m;j=1,2,3...n)\\\\

# 矩阵的乘法的几何意义

[210354]×[7689]=[222124276766]\begin{bmatrix}2&1\\\\0&3\\\\5&4\end{bmatrix}\times\begin{bmatrix}7&6\\\\8&9\end{bmatrix}=\begin{bmatrix}22&21\\\\24&27\\\\67&66\end{bmatrix} ​205​134​​×​78​69​​=​222467​212766​​

一个矩阵叉乘另一个矩阵，得到的还是一个矩阵，但是这个矩阵在图形学上被称作变换矩阵

即一个图形 进行 移动 + 旋转 + 缩放得到的是这个图形的变换矩阵，到最后这个样子需要将图形的原矩阵乘以变换矩阵

[1−102]×[12]=[−14]\begin{bmatrix}1&-1\\\\0&2\end{bmatrix}\times\begin{bmatrix}1\\\\2\end{bmatrix}=\begin{bmatrix}-1\\\\4\end{bmatrix} ​10​−12​​×​12​​=​−14​​

一个矩阵与一个列向量相乘，得到的是我们想要的最后结果，即这个顶点变换之后的坐标

在我们平时的实际开发中，我们不可能只单单对一个物体进行单个变换（移动，旋转，缩放），我们开始着手一个项目的时候往往是图形的复合变换，即包括图形的移动，旋转，缩放

在复合变换中我们采用的是列向量左乘

[2101]×[1−102]×[12]=[24]\begin{bmatrix}2&1\\\\0&1\end{bmatrix}\times\begin{bmatrix}1&-1\\\\0&2\end{bmatrix}\times\begin{bmatrix}1\\\\2\end{bmatrix}=\begin{bmatrix}2\\\\4\end{bmatrix} ​20​11​​×​10​−12​​×​12​​=​24​​

矩阵相乘的一个好处，我们可以除了最右边的向量不进行计算，先把左边的旋转，移动，缩放等先计算得到一个最终的变换矩阵，这时我们再拿最右边的向量与这个最终变换矩阵相乘得到最终结果
